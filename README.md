## ⚡ Quickstart

### Step 1: Install dependencies

```bash
# Circom circuit dependencies
cd src/circuits && npm install && cd ../..

# Node.js verifier dependencies
cd src/verifier && npm install && cd ../..
```

### Step 2: Build the Rust issuer

```bash
cd src/issuer
cargo build --release
cd ../..
```

### Step 3: Set up the circuits (one-time trusted setup)

```bash
cd src/circuits
chmod +x setup.sh
./setup.sh full_predicate
cd ../..
```

### Step 4: Run the full demo

```bash
chmod +x demo.sh
./demo.sh
```

## 👥 Member-Specific Quickstart

### Member 1 — Rust Dev (Issuer)

```bash
cd src/issuer

# Build
cargo build --release

# Issue a credential
cargo run --release -- issue --attributes "age=25,income=600000,smoker=0" --output credential.json

# Generate circom input for a proof
cargo run --release -- prove --credential credential.json --attribute age --threshold 18 --output input.json
```

### Member 2 — Circuit Builder (Circom)

```bash
cd src/circuits
npm install

# Generate a valid test input (IMPORTANT: run this first!)
node generate_input.js --value 25 --threshold 18

# Set up the circuit
./setup.sh full_predicate

# Generate a proof
./prove.sh input.json full_predicate
```

### Member 3 — Glue + Demo (Node.js)

```bash
cd src/verifier
npm install

# Verify a proof
node verifier.js --proof ../circuits/build/proof.json --public ../circuits/build/public.json --vkey ../circuits/build/verification_key.json --name "Bank"

# Run collusion demo (works even without circuits set up — uses simulation)
node collusion_demo.js

# Run benchmarks (requires circuits to be set up)
node benchmark.js --iterations 10
```

### Member 4 — Writer

- Design document template: `docs/design_document.md` — fill in benchmark numbers and create the diagram
- Architecture diagrams: `docs/architecture_diagram.md` — copy Mermaid to draw.io / Excalidraw
